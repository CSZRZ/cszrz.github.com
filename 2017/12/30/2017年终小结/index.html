第一次写年终小结，为啥呢，因为这年好像有点事情可以写了，啊哈哈哈。

一点总结

游戏

其实也没什么内容可以写的，不是优秀的人，也没做出什么优秀的事，可以说道说道。不过还是挺开心的，这一年最大的收获应该就属自己终于对游戏没有欲望了，还记得年初的时候，那阵子 LOL 玩的非常的疯狂，刚好工作上也没什么事，天天回家就是车队开黑，后来，开黑的小伙伴们，该上班的都上班了。自己排位的分数也很快打上去了，之后就感觉索然无味了，一个是达到了自己的心理预期，再高的段位不想上，另一个最主要的还是来源于自己的焦虑感吧。随着年纪的增大，周围的小伙伴，要么谈婚论嫁，要么就变的优秀，工资越来越高，平台也越来越高，可见的未来，自己只会和他们的差距越拉越大，看看自己女朋友也没有，工作上也没什么进展。真的就是突然之间对游戏失去了乐趣，再后来把为了玩游戏而特地组装的台式机送给了我姐。

工作

工作的话，真的是有种虚度了一年的感觉，大部分的时间可能都比较闲，自己也没有目标。划水，看看技术博客，时间就这样过去了，以至于到现在，除了下半年做的项目，真的想不起来有做的什么。上半年大部分还是处于颓废之中吧，浪费了大把大把的光阴，这些时光真的很难得，自己应该利用起来好好的整理知识的。不过还有另一些原因，偶尔领导需要你研究这个研究那个，或者做了一些要和别人对接的项目，其实这种项目真的没啥意义，沟通协调别人的成本是远远大于沟通成本的。嗯，说道沟通，随着待在公司的时间越长，感觉越发觉得某些缺点，尤其是和别人的沟通上，很多人真的是超出自己一点点的事，或者在模糊边界上的事，都会推三堵四，这点真的让我很不爽，虽然这些人和事到处都是，公司的性质决定了在实际开发中必然出现这样那样额的问题，这时候需要的是大家都有一个愿意沟通，愿意协调的心态才行，而不是生怕自己会多做一点事，哪怕这些真的是微不足道的事。

总的来说，工作上面对自己不满意，虚度了光阴，技术水平也是原地踏步。对公司也不满意，这点主要是和别人合作上的事，很难受，公司的氛围还是很轻松的，但是毕竟在公司里面是做事的，真的越来越体会到技术不是最大的障碍，最大的障碍从来都是人与人之间的。

生活

今年除了戒了游戏，还有一个就是生活上的改变吧，因为工作性质，所以很少需要加班，自己也有大量的时间，快年底的时候，终于做了些改变，开始阅读了，我想这真的对于我来说是一大进步，今年完整的看完的书没有几本，零零散散的看了一部分或者开头的很多。

小说

  • 追风筝的人
    微信读书上看的，感觉还不错，对于宗教，人性的思考吧

  • 白夜行
    同上，也是微信读书上看的，看完这些真的，感觉愈发需要规范自己的言行，因为自己的言行真的会给别人带来很多的苦难。

  • 解忧杂货铺
    其实是最早开始看的吧，看到知乎上很多东野圭吾的称赞，就去看了，个人原因吧，其实看了一半，真的无感，没有吸引我的地方

  • 腾讯传
    也只是看了个开头吧，在车上看的

小说基本都是在微信读书上看的，对微信读书这款App印象真的非常好。

技术

看了大量入门的教程，C， Python， Go， Swift，也是比较迷茫，同时也是为了打分时间，看了不少语言的入门语法，入门都是挺简单的，但是不用真的很快就忘了，现在业绩只有个大概印象。

同时，年底的时候，也开始尝试阅读开源库了，深感自己的水平不够啊，基础技术不够扎实，啃起来很费劲，都没看完几个。。

技术书籍方面其实有看不少吧，看完的，看一点的，看一部分的。买了很多的书，自己也看了不少的书，但是没有总结，照理说看了不少应该更有技术自信才对，但是仔细一想，却想不起来究竟学到了哪些，应该是看的不够认真吧，而且技术类的书籍真的需要反复阅读

  • 图解 HTTP 看完

  • 图解 TCP/IP 没看完

  • JS 高程 没看完

  • JS 语言精粹 粗略看完

  • 你不知道的 JS 没看完

  • CSS 权威指南 粗略看完

  • C 语言 粗略看完

  • 程序员的自我修养 这本是真的很好，可惜太难，只看了 1/3 明年需要好好看看

  • CSAPP 讲操作系统的，真的需要看，可惜页只看了一点。看不下去了

  • OC 52 个 tips 很好看，需要反复的阅读，写出高质量的代码

  • OSX 高级编程 也是很好的书，需要反复的看，思考

  • iOS UI 捷径 教你如何使用 IB 构建项目的，对于我这种的单兵来说，诱惑还是很大的，奈何实在没有自信新项目这样构建，也是懒惰，也是担心也爬不起来的坑耽误工期。

还有其他一些零散的没看完的书，也懒得列的，列出来也没啥意义，很多时候真的只是缓解下自己的焦虑感。

另外就是开始写博客了，博客老早就搭建好，写了 2 篇就弃了，因为感觉就是笔记，或者自己做的基本上是搬运工，没有什么高质量的，但是现在觉得还是很有必要的，不为给别人看，只为培养自己的习惯。

总结

啰嗦了这么多，实在不想再废话了，等会还要收拾东西回家。总的概括就是 虚度大半，开始惊醒，尝试改变，培养习惯!，明年希望坚持阅读，还是哟收获的，另外技术嘛，光看没有，还是得多 codig，多读开源库，明年主要发力应该是前端吧，感觉前端空间越来越大，然后希望能再用 Pyhton 或者 Go 写一下后端，学一门数据库，总的来说，明年有很多想做的，今天开始有改变了，明年希望都能落到实处，等到明年年底希望是一个丰收的一年~~~
