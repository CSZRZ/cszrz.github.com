MySql 安装可以通过 dmg 安装包来安装，也可以通过命令行来安装，这里的话我选择安装包安装，版本的话有 5.5，5.6，5.7，8.0，这里直接用最新的 8.0

安装 MySql

直接参考 MySql 的官方文档，下载地址
按步骤点就完事了，这就是安装包的方便了，哈哈（这里会在安装的时候要求输入数据库密码 对应 root 用户）

设置环境变量

这里根据自己情况在对应 shell 配置文件中配置，一般是 bash_profile 或者 zshrc 文件

一些基础的操作

  1. 登录 mysql 的 root 用户 ： mysql -u root -p
  2. 修改密码：set password = password(‘123’);
  3. 显示所有数据库：show databases;
  4. 新建数据库：create database mybatis_test;
  5. 删除数据库：drop database mybatis_test;
  6. 使用 mysql 数据库：use mysql;
  7. 显示数据中所有的表：show tables;
  8. 创建表：create table user(uid int(10) not null,username varchar(20) not null,password varchar(20),primary key(uid));
  9. 显示创建的表信息：desc user;
  10. 重命名表名：rename table user to usertable;
  11. 删除表：drop table usertable;
  12. 退出：exit; 或者 quit;
