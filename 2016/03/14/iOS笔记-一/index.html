关于UITableview

默认的样式UITableViewStylePlain,section的header和footer会有悬停的效果，且默认的背景色为白色
还有一种样式为UITableViewStyleGroup,默认的背景色是淡灰，选择这个样式的时候，需要为section的header设置值，默认情况下会有20几的高度的header，不需要的情况下需要设置为0.001(设置为0无效)

比如：

  • (CGFloat)tableView:(UITableView *)tableView heightForHeaderInSection:(NSInteger)section {
    if (section < 4)
    {
    return 0.001;
    }
    else
    {
    return 40;
    }
    }
