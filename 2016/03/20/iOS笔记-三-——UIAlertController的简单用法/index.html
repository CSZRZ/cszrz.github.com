UIAlertController的简单用法

iOS9.0API中放弃了UIActionSheet和UIAlertView2个类，取而代之的是UIAlertController类（有点像android中的DialogFragment），据说UIAlertController类在适配上很出色，另外API很友好，似乎UIActionSheet和UIAlertView非常繁琐，所以就直接学习一下这个“新类”的使用啦、

初始化

  • (instancetype)alertControllerWithTitle:(NSString )title message:(NSString )message preferredStyle:(UIAlertControllerStyle)preferredStyle
  • title:标题
  • message:细节信息
  • preferredStyle:样式，是action sheet还是alert

样式

  • UIAlertControllerStyleActionSheet
  • UIAlertControllerStyleAlert

按钮

  • UIAction

初始化：标题 + 样式（取消键、默认、示警键）+block

  • (instancetype)actionWithTitle:(NSString )title style:(UIAlertActionStyle)style handler:(void (^)(UIAlertAction action))handler

初始化后直接添加到UIAlertController

  • (void)addAction:(UIAlertAction *)action;

push

初始化结束后记得

[self presentViewController:alert animated:YES completion:nil];

示例代码

之前写过了，就直接贴过来了。

一个常规的alert view样式（1个文本+2个按钮）

UIAlertController *alert = [UIAlertController alertControllerWithTitle:@"提示框" message:@"这是一个窗口" preferredStyle:UIAlertControllerStyleAlert];
           UIAlertAction *cancle = [UIAlertAction actionWithTitle:@"取消" style:UIAlertActionStyleCancel handler:^(UIAlertAction * _Nonnull action) {
               NSLog(@"选择了取消按钮");
           }];
           UIAlertAction *ok = [UIAlertAction actionWithTitle:@"确定" style:UIAlertActionStyleDefault handler:^(UIAlertAction * _Nonnull action) {
               NSLog(@"选择了确定按钮");
           }];
           [alert addAction:cancle];
           [alert addAction:ok];

           [self presentViewController:alert animated:YES completion:nil];

一个用户登录样式的alert view

如果需要同时满足多个条件才能出发按钮，则需要添加观察者，然后记得移除观察者

UIAlertController *alert = [UIAlertController alertControllerWithTitle:@"文本对话框" message:@"这是一个窗口" preferredStyle:UIAlertControllerStyleAlert];
            [alert addTextFieldWithConfigurationHandler:^(UITextField * _Nonnull textField) {
                textField.placeholder = @"登陆";
                [[NSNotificationCenter defaultCenter] addObserver:self selector:@selector(alertTextFieldDidChange:) name:UITextFieldTextDidChangeNotification object:textField];
            }];
            [alert addTextFieldWithConfigurationHandler:^(UITextField * _Nonnull textField) {
                textField.placeholder = @"密码";
                textField.secureTextEntry = YES;
                [[NSNotificationCenter defaultCenter] addObserver:self selector:@selector(alertTextFieldDidChange:) name:UITextFieldTextDidChangeNotification object:textField];
            }];
            [alert addTextFieldWithConfigurationHandler:^(UITextField * _Nonnull textField) {
                textField.placeholder = @"name";
            }];
            [alert addTextFieldWithConfigurationHandler:^(UITextField * _Nonnull textField) {
                textField.placeholder = @"age";
            }];

            UIAlertAction *ok = [UIAlertAction actionWithTitle:@"好的" style:UIAlertActionStyleDefault handler:^(UIAlertAction * _Nonnull action) {
                UITextField *login = alert.textFields.firstObject;
                UITextField *password = [[alert textFields] objectAtIndex:1];
                UITextField *name = [[alert textFields] objectAtIndex:2];
                NSLog(@"name===================%@", name.text);
                [[ NSNotificationCenter defaultCenter] removeObserver:self name:UITextFieldTextDidChangeNotification object:nil];

            }];
            ok.enabled = NO;
            [alert addAction:ok];

            [self presentViewController:alert animated:YES completion:nil];


- (void)alertTextFieldDidChange:(NSNotification *)notification
{
UIAlertController *alert = (UIAlertController *)self.presentedViewController;
if (alert) {
    UITextField *login = alert.textFields.firstObject;
    UITextField *password = [[alert textFields] objectAtIndex:1];
    UIAlertAction *ok = alert.actions.lastObject;
    ok.enabled = login.text.length > 2 && password.text.length > 1;
}
}

ActionSheet样式的

一个上拉菜单（不可以添加UITextField），默认点击菜单以外的区域，菜单会消失

UIAlertController *alert = [UIAlertController alertControllerWithTitle:@"保存或删除数据" message:@"删除" preferredStyle:UIAlertControllerStyleActionSheet];
            UIAlertAction *cancle = [UIAlertAction actionWithTitle:@"取消" style:UIAlertActionStyleCancel handler:^(UIAlertAction * _Nonnull action) {
                NSLog(@"取消");
            }];
            UIAlertAction *delete = [UIAlertAction actionWithTitle:@"删除" style:UIAlertActionStyleDestructive handler:^(UIAlertAction * _Nonnull action) {
                NSLog(@"删除");
            }];
            UIAlertAction *ok = [UIAlertAction actionWithTitle:@"确定" style:UIAlertActionStyleDefault handler:^(UIAlertAction * _Nonnull action) {
                NSLog(@"确定");
            }];
            [alert addAction:cancle];
            [alert addAction:delete];
            [alert addAction:ok];

            [self presentViewController:alert animated:YES completion:nil];
